# Banner image slots

Put production banner photos in this folder. Use `.webp` when possible and `.jpg` for Android-safe photographic banners. Keep `.png` only for prepared artwork that has been tested in an Android release build.

Recommended image size: `1600x1000 px` or `1600x900 px`; minimum `1200x750 px`; target file size `300-500 KB`.

Required names:

- `hero-home.jpg`
- `hero-services.jpg`
- `hero-universities.jpg`
- `hero-country.jpg`
- `hero-city.jpg`
- `hero-university.jpg`
- `hero-program.webp`
- `hero-application.jpg`
- `hero-visa.jpg`
- `hero-tours.jpg`
- `hero-admission.jpg`
- `hero-settings.webp`
- `hero-profile.webp`

Photos should avoid small text, important objects near edges, and very bright areas because hero text is rendered on top with a red/dark-red overlay.

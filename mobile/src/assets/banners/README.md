# Banner image slots

Put production banner photos in this folder. Use `.webp` when possible, `.jpg` is acceptable.

Recommended image size: `1600x1000 px` or `1600x900 px`; minimum `1200x750 px`; target file size `300-500 KB`.

Required names:

- `hero-home.webp`
- `hero-services.webp`
- `hero-universities.webp`
- `hero-country.webp`
- `hero-city.webp`
- `hero-university.webp`
- `hero-program.webp`
- `hero-application.webp`
- `hero-visa.webp`
- `hero-tours.webp`
- `hero-admission.webp`
- `hero-settings.webp`
- `hero-profile.webp`

Photos should avoid small text, important objects near edges, and very bright areas because hero text is rendered on top with a red/dark-red overlay.

# Banner image slots

Put production banner photos in this folder. Use `.webp` when possible, `.jpg` is acceptable, and `.png` is supported for prepared artwork.

Recommended image size: `1600x1000 px` or `1600x900 px`; minimum `1200x750 px`; target file size `300-500 KB`.

Required names:

- `hero-home.png`
- `hero-services.png`
- `hero-universities.png`
- `hero-country.jpg`
- `hero-city.jpg`
- `hero-university.jpg`
- `hero-program.webp`
- `hero-application.jpg`
- `hero-visa.png`
- `hero-tours.jpg`
- `hero-admission.png`
- `hero-settings.webp`
- `hero-profile.webp`

Photos should avoid small text, important objects near edges, and very bright areas because hero text is rendered on top with a red/dark-red overlay.
